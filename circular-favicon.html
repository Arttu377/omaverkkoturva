Ympyrän muotoinen favicon - Kappaleen teksti (52)

Tämä on esimerkki miten tehdä ympyrän muotoinen favicon kuvasta "Kappaleen teksti (52).png"

16x16 (favicon.ico)
32x32 (favicon)
48x48 (apple-touch-icon)
64x64
128x128

Käyttöohjeet:

1. CSS-ympyrä (suositteeltu)

Käytä CSS:ää tehdäksesi kuvasta ympyrän:


.circular-logo {
    width: 40px;
    height: 40px;
    border-radius: 50%;
    overflow: hidden;
}

.circular-logo img {
    width: 100%;
    height: 100%;
    object-fit: cover;
    border-radius: 50%;
}

2. React-komponentti

Käytä CircularLogo-komponenttia:


import { CircularLogo } from './components/CircularLogo';

<CircularLogo 
    imageUrl="/kuvapankki/Kappaleen teksti (52).png"
    alt="Ympyrän muotoinen logo"
    size={40}
    borderWidth={2}
    borderColor="#ffffff"
/>

3. Favicon-käyttö

Voit käyttää ympyrän muotoista kuvaa faviconina:


<link rel="icon" href="/kuvapankki/Kappaleen teksti (52).png" type="image/png" />

Huomio: Favicon näkyy ympyrän muotoisena vain jos selain tukee sitä. Useimmat selaimet näyttävät faviconin neliön muotoisena.
